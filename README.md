# Lowland Programming Language

Lowland is a statically-typed, interpreted programming language designed for performance and ease of use.

## Why use LowLand?
- ⚡⚡ Fast Developing 
- ✈️✈️ Ease of Syntax
- ✔️✔️ General Purpose
